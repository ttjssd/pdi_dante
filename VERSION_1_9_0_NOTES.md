# PDI Backoffice v1.9.0 준비 메모

## 후보 작업

- 콘솔 도구 검색 고도화
  - 최근 사용 도구 저장
  - 검색 결과 키워드 하이라이트
  - `Ctrl + K` 빠른 검색 포커스

- 탁송 보조 툴 확장
  - 실제 기사 문의 사례 기반 유형 추가
  - 답변 문구 즐겨찾기 또는 사용자 커스텀 템플릿

- 누락(재상품화) 가이드 확장
  - 항동센터/누락 가이드 간 교차 링크
  - 외부 Notion 참고 링크 연결

- 런처 개선
  - 프라이빗 모드 배경 프리셋 관리
  - 업무 모드/개인 모드 표시 옵션 정리

## 버전 원칙

- 다음 일반 기능 업데이트는 `1.9.0` 후보로 관리한다.
- 메이저 버전 `2.0.0`은 별도 요청 전까지 올리지 않는다.
