# PDI Backoffice 1.9.0

## 주요 변경

- 슬랙 일일 업무일지 원문 분석
- 일일 출고준비, 특이사항, 탁송 인계 수 자동 추출
- 항동 관리 업무 및 인사 항목 추출·수정·저장
- 금요일~목요일 기준 주간 리포트 생성
- 전주 대비 출고완료, 출고준비, 특이사항 증감 계산
- 일일 업무일지와 주간 리포트 복사
- localStorage 기록 수정·삭제
- JSON 데이터 내보내기·가져오기

## 데이터 저장

업무일지 데이터는 사용 중인 PC의 브라우저 저장소에만 보관됩니다.
정기적으로 JSON 내보내기를 사용해 별도 백업하는 것을 권장합니다.
